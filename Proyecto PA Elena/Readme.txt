Tarea: Programación Aplicada Universidad del Istmo, Desarrollo de Sitio Web 

Descripción
Este proyecto consiste en el desarrollo de un sitio web para la empresa ficticia "Time Care Services", que ofrece servicios de consultoría y desarrollo profesional. El sitio web consta de cuatro secciones principales: Inicio, Quiénes Somos, Productos y Contáctenos.

Contenido del Repositorio
- index.html: Página principal del sitio web que incluye el menú de navegación y una introducción a los servicios de Time Care Services.
- quienes_somos.html: Página que describe la empresa, su misión, visión y valores, junto con una foto del gerente.
- productos.html: Página que muestra los productos y servicios ofrecidos por Time Care Services, con sus respectivas descripciones.
- contacto.html: Página que contiene información de contacto de la empresa y un formulario de contacto.
- styles.css: Archivo CSS que contiene los estilos para el sitio web.
- script.js: Archivo JavaScript que contiene el código para mostrar el día de la semana en el encabezado.
- logo.png: Imagen del logo de Time Care Services.
- gerente_foto.jpg: Imagen Elena Obrejan gerente de la empresa.
- cover_image.jpg: Imagen de portada para las páginas.

Instrucciones de Uso
1. Clona este repositorio en tu máquina local.
2. Abre cualquier archivo HTML en tu navegador web para ver el sitio web.
3. Navega a través de las diferentes secciones utilizando el menú de navegación.
